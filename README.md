# ①課題名
クローゼット管理アプリ

## ②課題内容（どんな作品か）
- PHPを使って、洋服を登録・管理できるアプリを作成しました。
- メルカリのような「一覧・登録・編集・削除」ができる簡易的なEC風システムです。
- 画像アップロード機能も実装し、視覚的にアイテム管理できるようにしました。

## ⑤工夫した点・こだわった点
- 登録・一覧・編集・削除機能を実装しました。
- 画像アップロード機能（move_uploaded_file）を追加し、視覚的に管理できるようにしました。
- GETパラメータを使ったシーズン・カテゴリの絞り込み検索機能を実装しました。
- CSSでカードUI・フォームUIを整え、シンプルで見やすいデザインにしました。
